# TorusCycle

An incomplete Tron WebGL clone. [Preview it.][preview]

[![TorusCycle screenshot](screenshot.jpg)][preview]

## Build

    cd tools
    python2 build.py

## Run it

    cd ../dist
    python2 -m SimpleHTTPServer 1234

Go to [localhost:1234](http://localhost:1234).
